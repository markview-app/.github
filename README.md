# .github Repository

This is a special repository for the MarkView organization profile.

## Structure

- `profile/README.md` - Organization profile displayed on <https://github.com/markview-app>

## Setup Instructions

1. Create a new repository named `.github` in the markview-app organization
2. Copy the contents of this folder to that repository
3. The `profile/README.md` will automatically be displayed on your organization's profile page
