# MarkView

**Modern Markdown Viewer for Chrome, Edge & Chromium Browsers**

Transform your markdown files into beautifully formatted documents with syntax highlighting, Mermaid diagrams, KaTeX math, and powerful navigation features.

## ‚ú® Features

### üìù Enhanced Markdown Rendering

- **Full GFM Support** - Tables, task lists, strikethrough, footnotes, and more
- **Syntax Highlighting** - Beautiful code blocks for 180+ programming languages with 10+ customizable themes
- **Mermaid Diagrams** - Flowcharts, sequence diagrams, gantt charts with zoom controls and pan/drag support
- **KaTeX Math** - Beautiful mathematical equations with LaTeX syntax
- **Professional Image Viewer** - Lightbox with gallery navigation, zoom controls, pan/drag support, and intelligent lazy loading

### üé® User Experience

- **Multi-Language Support** - Interface available in 12 languages
- **Dark Mode** - Eye-friendly theme with customizable appearance
- **Interactive Table of Contents** - Auto-generated, collapsible TOC with smooth scrolling
- **Reading Progress Bar** - Visual indicator showing your position in the document
- **External Link Preview** - Visual indicators and preview icons for external links
- **Privacy-First** - All processing happens locally, no data collection

### üöÄ Productivity Features

- **Live Auto-Refresh** - Real-time preview updates when files change - perfect for writing and editing
- **Raw Markdown Toggle** - Instantly switch between rendered and source view with one click
- **Folder Browser** - Navigate your local markdown project structure with ease
- **Folder Bookmarks** - Save and quickly access frequently used directories
- **Recent Documents** - Quick access to recently opened files from the popup menu
- **Word Count & Reading Time** - Live statistics showing word count, character count, and estimated reading time
- **Export to DOCX & HTML** - Perfect formatting preservation for sharing and collaboration
- **Print-Friendly Layouts** - Clean, professional printing without UI elements
